About Me

I’m a young, passionate, aspiring designer and web developer. I’m learning new things about design and development everyday. I love doing mockups of sites and hand coding them, making sure I change my approach to it each time. I believe practice really does make perfect.

Here are some things I believe I'm good at:-

  • HTML 5 & Jade
  • CSS & SASS
  • Responsive Web Design
  • Photoshop
  • Illustrator

Design Process

My design process always begins with me asking myself four very important questions. Who is the target audience? What is important to the audience? Who is the owner of the website and what is important to them? Once these questions have been answered, we can then define the features of the website.

I then move on to wireframing the website. I like to use old fashioned pencil and paper for this. I’ll then do some mockups of each idea to see how it would look like in the browser.

The next step would be to begin development. For this I like using Github’s text editor Atom and for compiling and minifying my SASS, Jade, HTML, CSS and Javascript I like to use a multi-platform application called Prepros

After I have finished developing the site, testing begins!
